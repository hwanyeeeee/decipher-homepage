Features

주요 기능

메시지 분석

Message.sqlite과 Talk.sqlite에서 모든 채팅 메시지를 추출하고 분석합니다.

삭제 메시지 복원

"나에게만 삭제"와 "전체 삭제"를 정확히 구분하고, 잔존 데이터를 기반으로 삭제된 메시지를 복원합니다.

사용자 ID 매핑

친구 데이터베이스와 교차 참조하여 사용자 ID를 실제 이름과 매핑합니다.

암호화 백업 지원

iTunes 암호화 백업과 비암호화 백업을 자동으로 처리합니다. 오픈 채팅 복호화에는 키체인이 포함된 암호화 백업이 필요합니다.

💬

오픈 채팅 분석

별도로 암호화된 OpenChatMessage.sqlite 데이터베이스를 분석합니다.

🔑

키체인 추출

iOS 키체인에서 오픈 채팅 분석에 필요한 키를 자동으로 추출합니다.

📊

대화 이벤트

입장, 퇴장, 강퇴, 초대 등 대화 이벤트를 사용자 이름과 함께 표시합니다.

🔍

DB 조인 분석

오픈 채팅 DB의 복잡한 관계를 자동으로 해석하고 연결합니다.

🖼️

이미지 자동 매핑

이미지 파일을 원본 메시지(attachment 레코드)와 자동으로 매핑하여 표시합니다.

🔎

이미지 뷰어

썸네일 미리보기와 원본 이미지 확대 뷰어를 제공합니다.

📁

채팅 디렉토리 복원

Manifest.db 기반으로 백업 구조를 해석하고 채팅 디렉토리를 재구성합니다.

📎

다양한 메시지 타입

사진, 동영상, 음성, 위치, 연락처, 이모티콘 등 모든 메시지 타입을 처리합니다.

🔍

강력한 검색

메시지, 채팅방, 친구 검색. 하이라이트 표시와 디바운스로 빠른 검색.

📊

분석 실패 검사

분석에 실패한 메시지를 채팅방별로 정리하여 보여줍니다.

💾

SQLite 내보내기

분석 결과를 독립 SQLite 데이터베이스로 내보내 안전하게 보관합니다.

⏱️

타임라인 재구성

복잡한 검색 필터링과 함께 대화의 시간순 타임라인을 재구성합니다.
