# DECIPHER Website

DECIPHER 프로젝트 웹사이트. iOS 메신저 포렌식 도구 시리즈(iKakao, iLine, iTelegram) 소개 페이지.

## Tech Stack

- HTML + Tailwind CSS (CDN) + Vanilla JavaScript
- 정적 웹사이트 (빌드 도구 없음)
- 한국어/영어 콘텐츠
