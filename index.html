Features

왜 DECIPHER인가?

포렌식 전문가를 위한 신뢰할 수 있는 도구

포렌식 무결성

원본 데이터를 절대 변경하지 않는 읽기 전용 분석. 해시 검증으로 증거 보전을 보장합니다.

강력한 데이터 분석

암호화된 백업 데이터도 분석 가능. iOS iTunes 암호화 백업을 완벽하게 지원합니다.

삭제 메시지 복원

삭제된 메시지를 복원하고, "나에게만 삭제"와 "전체 삭제"를 정확하게 구분합니다.

미디어 복원

이미지, 동영상 등 미디어 파일을 자동으로 매핑하고 원본 미리보기를 제공합니다.

직관적 인터페이스

메신저 원본과 유사한 UI로 자연스럽게 분석. 실시간 진행률, 검색, 필터 기능 제공.

독립 실행

단일 실행 파일로 별도 설치 없이 바로 사용. 분석 결과는 독립 DB로 안전하게 저장.

How It Works

간단한 3단계

01

백업 선택

iTunes 백업 폴더를 선택하고 백업 비밀번호를 입력합니다.

02

자동 분석

DECIPHER가 자동으로 데이터를 추출하고 분석합니다.

03

결과 확인

메시지, 미디어, 메타데이터를 직관적인 UI로 확인합니다.
